- [restyle](https://github.com/Shopify/restyle)

# :wrench: Rodando o projeto

```bash
git clone git@github.com:the-spanish-guy/boilerplate-mobile.git boilerplate

cd boilerplate

yarn instal
```

ou clique em `use this template`
